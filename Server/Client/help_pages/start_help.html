Welcome to ClaferMoo Visualizer


--Click anywhere outside help windows to hide.

In this tool, you can visualize and explore the set of non-dominated optimal variants (called Pareto front) computed by Clafer Multi Objective Optimizer (ClaferMoo).

Press the '?' button of each window to see specific help. In general:

-- Use the input view to select a .cfr model and optimize it.

-- Use the Bubble Front Graph view to visually explore the Pareto front.

-- Use the Feature and Quality Matrix view to filter the instances by selecting and eliminating features.

-- Use the Variant Comparer view to perform tradeoff analysis of a set of selected variants.
